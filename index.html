BizTech

A blog website built with React, Tailwind CSS and Firebase. Users can register, write blog posts, like posts, and comment on blog posts. The site allows users to edit and delete comments and delete posts. To enhance the user experience, users can search and select images from the Unsplash API to include at the top of their posts. You can also view posts by author, topic, or user.

StockOptima

A fullstack inventory management website built with React, Tailwind CSS, Node, Express and MongoDB. Allows users to create, edit and delete products displayed in a table with search and filter functionality, a stats page to give you a breakdown of all your products, and a profile page where you can update your name, profile image, and email address.
NOTE: Site can take up to 30 seconds to initially load because I'm using a free instance to host stockOptima.

The application provides a search feature, allowing users to quickly find specific glasses based on their titles. Can add products to the cart, then use checkout with form validation. Gets product data from a serverless function connected to an airtable.
